Apples and Bananas

The apples and bananas methodology is simple yet effective. By using easily memorizable objects we can teach algebra to students, such as yourself! Now I'm sure you're probably saying right now "I already know about apples and bananas!", sometimes all we need to do is go back to basics. If I had 3 apples which added together to make 30, what is the value of 1 apple? In this situation an apple is called a variable. A variable is something which holds a number or equation value. To solve this equation all we have to do is divide the sum of the apples by how many apples there are, as seen in this equation: $$ apple = {cost \over quantity} = {30 \over 3} = 10 $$ This equation told us that each apple costs $10, how easy was that?



Now I am going to integrate letters into the mix. Instead of using numbers in an equation, we can use something called a 'variable'. In the above example we use 'apples and bananas' to replace numbers, the apple is a variable because it held a value of $10! Let us start simple. We will say there are 20 students going on a trip to the fossil museum, and each student needs to pay 'x' amount of dollars. If the combined payments equal $100, then what is 'x'?Now, I know there are a lot of words in this equation but really you only need to highlight the key numbers and instructions. In this question we need to find what each student paid individually, if we think in equation terms we can find this equation: $$ 100 = 20 n $$
