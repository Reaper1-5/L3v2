Basic Arithmetic

Addition with letters : \(2a + 6a = 8a\)


Subtraction with letters : \(7a - 2a = 5a\)


Multiplication with letters : \(a\times b = ab \)


Dividing with letters : \(a \div b = \frac{a}{b} \)


Powers


When multiplying, if the bases are the same we add the powers : \( a^5\times a^2 = a^{5+2} = a^7 \)


When dividing, if the bases are the same we subtract the powers : \( \frac{a^5}{a^2} = a^{5-2} = a^3 \)


When there is a power inside and outside of the brackets, we multiply the powers : \( (a^5)^3 = a^{3\times5} = a^{15}\)


When there is a power outside of the brackets, we distribute the power to all numbers/variables : \( (ab)^8 = a^{8}b^{8} \)
